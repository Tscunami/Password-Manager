# Password Manager

---

Offline Password Manager storing:
* **Domain**
* **Username / Email**
* **Password**   

---
### Features
* **Generate a password** generates a new strong 
  password and copy it  
* **Save** checkbox will store email for next use
* **Show Data** button opens data file 
with stored passwords
